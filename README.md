# Interactive Perspective Transformer (交互式透视变换校正工具)

这是一个基于 **PyQt6** 和 **OpenCV** 构建的轻量级桌面应用程序，专门用于修复由于拍摄角度倾斜导致的图像失真。它能够将倾斜的物体（如文档、海报、画框）还原为规整的正面矩形。

---

## ✨ 核心特性

* **智能选点交互**：通过鼠标点击图片即可定义四个关键顶点，支持实时撤销重置。
* **自适应 UI 设计**：界面支持动态缩放，自动处理屏幕像素坐标与图像真实像素坐标的映射。
* **自动尺寸计算**：基于欧几里得距离算法，自动推导目标矩形的最优分辨率，防止图像拉伸。
* **中文路径兼容**：采用 Numpy 编解码方案，处理包含中文路径的文件时不会报错。
* **高精度处理**：使用 OpenCV 的透视变换矩阵算法，保留图像细节。
